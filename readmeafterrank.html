Rank
When the game ends, you will receive a sixteen-digit numeric rank code. This code is only valid for your account name. You can enter your code in a new game instead of creating a hero, restoring your previous hero configuration selections. at the end of each game in order to accumulate more rank experience for your hero.

Heroes must achieve the rank of National Hero before they can participate in a Nightmare game which is where the real challenge begins. Only Legendary Heroes can attempt Extinction, the experts' difficulty level.

Earning Rank Exp.
You can earn rank experience whether you're online (battle.net) or offline (local network), and with any number of players in the game. You cannot earn rank experience if you create the game in single-player. But you can solo and earn rank experience if you simply create a Local Network game where you are the only player.

Medals
When the game ends, medals will be awarded if the team accomplished certain achievements. No medal will be awarded to more than one hero in a game, and no hero may earn more than one medal in a game. If you receive a medal, be sure to record your new rank code. Medals are primarily a source of prestige; medal bonuses are only activated when attempting Extinction as a team or Nightmare alone.
